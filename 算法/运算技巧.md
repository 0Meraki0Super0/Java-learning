### 位运算

```java

1. x << y：表示 将 x 向左移动 y 位 
 - x >> y: 表示将 x 向右移动 y 位.
对于一个 32 位的整数，通常从最低位（即第 0 位）到最高位（即第 31 位）进行编号 i 的取值范围是从 0 到 31，表示遍历整数的每一位，其中 0 表示最低位，31 表示最高位。
2. x&(x-1):消去最低位1。2的幂次方得0
3. `|`：按位或运算符。用于对两个数的对应位进行或操作。如果两个对应位中至少有一个为 1，则结果为 1
	可用于拼接新的二进制串 ans=ans|(1<<i)
   `^`：按位异或运算符
4. Integer.bitCount(x):计算x的二进制1的个数
5. -1的二进制11111···,它和任何数AND值都为那个数。可作为&结果的初始值
```
